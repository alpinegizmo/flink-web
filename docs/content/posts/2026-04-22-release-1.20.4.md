The Apache Flink Community is pleased to announce the fourth bug fix release of the Flink 1.20 series.

This release includes 41 bug fixes, vulnerability fixes, and minor improvements for Flink 1.20.
Below you will find a list of all bug fixes and improvements (excluding improvements to the build infrastructure and build stability). For a complete list of all changes see:
[JIRA](https://issues.apache.org/jira/secure/ReleaseNote.jspa?version=12356275&projectId=12315522).

We highly recommend all users upgrade to Flink 1.20.4 or higher.

The Apache Flink Community is pleased to announce the second bug fix release of the Flink 1.20 series.

This release includes 16 bug fixes, vulnerability fixes, and minor improvements for Flink 1.20.
Below you will find a list of all bugfixes and improvements (excluding improvements to the build infrastructure and build stability). For a complete list of all changes see:
[JIRA](https://issues.apache.org/jira/secure/ReleaseNote.jspa?projectId=12315522&version=12355933).

We highly recommend all users upgrade to Flink 1.20.3 or higher.
